Linux-挂载windows共享盘

2019-10-09
  1. 确保windwos下共享盘已经开启，并且可以通过用户名密码访问
  2. 使用命令 apt-get install cifs-utils 安装cifs协议
  3. 使用命令 mkdir /mnt/win7 创建挂载目录
  4. 使用命令 mount -t cifs -o username=xxxx,password=xxxx //xxx.xx.xx.xx/e$ /mnt/win7 进行挂载
  5. 进入目录 /mnt/win7 查看是否挂载成功
